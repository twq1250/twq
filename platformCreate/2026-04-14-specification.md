# 开放平台应用管理后台 - 项目规格文档

## 一、项目概述

### 1.1 项目简介

本项目是一个开放平台的应用管理后台，用于管理企业内部应用的开发、配置和发布流程。

### 1.2 技术栈

- **框架**: React 18
- **构建工具**: Vite
- **路由**: React Router
- **UI 组件库**: Ant Design 4
- **样式**: CSS Modules (less)
- **状态管理**: React Hooks (useState, useEffect, useMemo)

## 四、布局结构

### 4.1 整体布局

项目采用双模式布局设计：

**首页布局 (pathname === '/')**

```
┌─────────────────────────────────────────────────────────────┐
│                      Header (64px)                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│                     <Outlet />                              │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**详情页布局 (pathname !== '/')**

```
┌─────────────────────────────────────────────────────────────┐
│                      Header (64px)                          │
├─────────────────────────────────────────────────────────────┤
│                   AppInfoBar (50px)                         │
├──────────────┬──────────────────────────────────────────────┤
│              │                                             │
│   Sidebar    │               <Outlet />                    │
│   (220px)    │                                             │
│              │                                             │
└──────────────┴─────────────────────────────────────────────┘
```

### 4.2 尺寸常量

| 常量                     | 值     | 说明      |
| ---------------------- | ----- | ------- |
| HEADER\_HEIGHT         | 64px  | 顶部导航高度  |
| APP\_INFO\_BAR\_HEIGHT | 50px  | 应用信息栏高度 |
| SIDE\_PADDING          | 16px  | 侧边栏内边距  |
| PADDING                | 24px  | 内容区内边距  |
| SIDEBAR\_WIDTH         | 220px | 侧边栏宽度   |

### 4.3 高度计算公式

```javascript
// 详情页内容区高度
const contentAreaHeight = `calc(100vh - ${HEADER_HEIGHT}px - ${APP_INFO_BAR_HEIGHT}px - ${PADDING}px)`;
// 即: calc(100vh - 64px - 50px - 24px)

// 首页内容区高度
const contentAreaHeight = `calc(100vh - ${HEADER_HEIGHT}px - ${PADDING}px)`;
// 即: calc(100vh - 64px - 24px)
```

## 九、总结

本项目是一个功能完整的开放平台应用管理后台，包含以下核心模块：

1. **应用管理**: 首页展示应用列表，应用卡片点击进入详情
2. **基础信息**: 应用凭证管理、中英文信息配置、认证方式选择
3. **成员管理**: 添加/删除应用成员，角色权限配置
4. **应用能力**: 机器人、网页应用、小程序、小组件的添加和配置
5. **API 管理** ⭐: API 权限开通、管理，支持 SOA/APIG 两种认证类型
6. **事件配置** ⭐: 事件订阅配置，支持 MQS 和业务系统两种回调方式
7. **版本发布**: 版本创建、查看、删除管理

项目采用 React + Ant Design 4 技术栈，布局结构清晰，交互体验流畅，代码模块化程度高，便于维护和扩展。
