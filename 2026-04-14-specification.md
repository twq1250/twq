## 2. 页面布局与交互

### 2.1 路由结构

| 路由路径 | 页面组件 | 说明 |
|---------|---------|------|
| `/` | AppList | 首页(应用列表) |
| `/basic-info?appId=x` | BasicInfo | 凭证和基础信息 |
| `/members?appId=x` | Members | 成员管理 |
| `/capabilities?appId=x` | Capabilities | 添加应用能力 |
| `/capability-detail?appId=x&type=x` | CapabilityDetail | 能力详情 |
| `/api-management?appId=x` | ApiManagement | API管理 |
| `/events?appId=x` | Events | 事件配置 |
| `/version-release?appId=x` | VersionRelease | 版本发布 |
| `/version-release/form?appId=x&versionId=x` | VersionForm | 版本表单 |

### 2.3 详情页布局 (通用)

详情页采用统一的布局结构，由 Layout 组件根据路由路径判断是否为详情页:

**布局结构**:
```
┌─────────────────────────────────────────────────────────────┐
│                        Header (64px)                        │
├─────────────────────────────────────────────────────────────┤
│                    AppInfoBar (50px)                       │
├──────────────┬──────────────────────────────────────────────┤
│              │                                              │
│   Sidebar    │              Content Area                    │
│   (220px)    │         (calc(100vh - 114px))              │
│              │                                              │
└──────────────┴──────────────────────────────────────────────┘
```

**高度计算公式**:
- Header: 64px
- AppInfoBar: 50px
- 内容区内边距: 24px
- 侧边栏/内容区高度: `calc(100vh - 64px - 50px - 24px)` = `calc(100vh - 138px)`
